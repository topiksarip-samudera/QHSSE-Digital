# Prompt Continue Sequence

Continue Sequence. Kerjakan core berikutnya sesuai file sequence yang sedang aktif.

Aturan:
- Jangan lompat core.
- Jangan lanjut ke core berikutnya setelah selesai.
- Pastikan database, backend, frontend, permission, audit log, dan test selesai.
- Tulis `SELESAI CORE: <nama core>` di akhir.
