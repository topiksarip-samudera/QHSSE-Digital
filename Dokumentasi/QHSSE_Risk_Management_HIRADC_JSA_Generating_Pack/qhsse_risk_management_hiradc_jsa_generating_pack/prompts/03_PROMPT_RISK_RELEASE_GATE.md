# Prompt Risk Release Gate

Jalankan release gate Risk Management / HIRADC / JSA.

Cek:
- P0 = 0
- P1 = 0
- Tenant isolation PASS
- Permission backend PASS
- Risk calculation PASS
- HIRADC calculation PASS
- JSA calculation PASS
- Workflow PASS
- Audit log PASS
- Dashboard PASS
- Export PASS
- Regression PASS

Jika lulus tulis `RISK MANAGEMENT STABILIZED: GO`.
Jika tidak lulus tulis `RISK MANAGEMENT STABILIZED: NO-GO` dan daftar blocking issue.
