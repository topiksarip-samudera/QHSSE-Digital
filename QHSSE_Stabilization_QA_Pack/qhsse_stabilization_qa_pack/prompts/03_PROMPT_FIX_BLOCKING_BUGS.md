# Prompt Fix Blocking Bugs

Review BUG_REGISTER.md.

Fokus hanya bug P0 dan P1.

Untuk setiap bug:
1. Reproduce.
2. Cari root cause.
3. Fix.
4. Tambahkan test agar bug tidak muncul lagi.
5. Jalankan regression.
6. Update BUG_REGISTER.md dan FIX_LOG.md.

Jangan lanjut ke modul QHSSE Operational sebelum P0=0 dan P1=0.
