# Prompt Incident Release Gate

Jalankan release gate untuk Incident Management.

Cek:
- P0 = 0
- P1 = 0
- Tenant isolation PASS
- Permission backend PASS
- Workflow PASS
- RCA PASS
- CAPA PASS
- Attachment security PASS
- Notification PASS
- Audit log PASS
- Dashboard/report PASS
- Lint/test/build PASS

Jika lulus tulis:

`INCIDENT MANAGEMENT STABILIZED: GO`

Jika tidak lulus tulis:

`INCIDENT MANAGEMENT STABILIZED: NO-GO`

dan daftar blocking issues.
